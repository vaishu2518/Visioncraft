# VisionCraft

A web-based AI content creation platform built with Python and 
Streamlit. VisionCraft helps users edit and enhance images, generate 
thumbnails, and automatically create captions and hashtags for social 
media content.

## What it does
- Allows users to upload and edit images directly in the browser
- Enhances images for better visual quality
- Generates ready-to-use thumbnails
- Automatically creates engaging captions and relevant hashtags using AI

## Key Value
Combines multiple content-creation steps (image editing, thumbnail 
generation, caption/hashtag writing) into a single streamlined tool — 
saving creators time across multiple platforms and apps.

## Tech Stack
- Python
- Streamlit
- AI/LLM integration for caption and hashtag generation
- Image processing libraries

## How to run
1. Install dependencies (see requirements in the repo)
2. Run the Streamlit app: `streamlit run [your main file name].py`
3. Upload an image and generate enhanced visuals, thumbnails, captions, 
   and hashtags directly in the browser

## Use Case
Ideal for content creators, social media managers, or small businesses 
who need quick, polished visual content without manual editing or 
writer's block for captions.
